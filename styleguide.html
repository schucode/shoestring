Buttons

The button, button-small and button huge class can be applied to link, button, input type="button" We will also test that it can be applied to input type="submit" and <button>. By default, all of these buttons get the $button-default color from the vars.sass file. As you can see, the button-huge class fills it's container while the others are display inline-block.
